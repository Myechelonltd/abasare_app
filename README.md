# abasare_app
# Guidelines

## Clone the Repo

```
$ git clone https://github.com/Myechelonltd/abasare_app.git 
```

## Enter Directory

```
$ cd abasare_app
```
## install
```

$ npm i 
```

## Development

```
$ npm run dev
```

## Start Application

```
$ npm start
```
